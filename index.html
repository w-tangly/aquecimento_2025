Principais carreiras

Ciência de dados

O Cientista de Dados é responsável por estabelecer uma base de dados para executar análises. Com esse trabalho, as empresas conseguem entender o negócio, tomando as melhores decisões. 

Engenharia de software 

O profissional da área de engenharia de software é responsável por projetar o desenvolvimento de programas, aplicativos e sistemas

Desenvolvedor mobile

Desenvolve aplicativos e aplicações para celulares, tablets e outros dispositivos mobiles

Gestão da tecnologia da informação

A gestão de TI como podemos tem como objetivo o gerenciamento de hardwares e softwares, auxiliando também na organização.

Programador web

 O profissional dessa área é responsável por criar e desenvolver  sites, portais, fóruns, blogs, entre outras plataformas.
